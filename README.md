# Skin Cancer Detection AI

This project uses a Convolutional Neural Network (CNN) to detect skin cancer and other skin diseases using the HAM10000 dataset.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Download data:
   ```bash
   python src/data_setup.py
   ```

3. Train model:
   ```bash
   python src/train.py
   ```

4. Run App:
   ```bash
   streamlit run app.py
   ```
